# Internationalization (i18n)

## Adding a Translation

1. Open `src/translations/en.json`.
2. Add your new key-value pair:
   ```json
   {
     "my_new_feature": {
       "title": "My Feature Title"
     }
   }
   ```

## Usage in Code

Use the `i18n.t` helper to retrieve strings:

```javascript
import i18n from './lib/i18n'; // Adjust path as needed

const title = i18n.t('my_new_feature.title');
// Returns: "My Feature Title"
```

## Interpolation

You can pass variables to strings:

```javascript
// JSON: "welcome": "Hello {{name}}!"
i18n.t('welcome', { name: 'Alice' }); 
// Returns: "Hello Alice!"
```
